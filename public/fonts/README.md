# Opus Brand Fonts

Place the following Brokman font files in this directory:

- `BrokmanTrial-Light.otf` (weight: 300)
- `BrokmanTrial-Regular.otf` (weight: 400)
- `BrokmanTrial-Medium.otf` (weight: 500)
- `BrokmanTrial-Bold.otf` (weight: 700)

These are sourced from the OPUS Graphic Guidelines (February 2018 Update).
